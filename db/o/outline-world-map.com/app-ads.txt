<p>This website provides a set of different outline world map images. These pictures have by default gray scale or transparent continent, ocean and outline (border) areas. You can use also our world map painting tool to create from them colored world map images, choosing such colors, which you like for these map areas.<br />
<br />
In other words, this website has two main parts:<br />
1. outline world map images, which land, water and border areas are white, black, gray or transparent;<br />
and<br />
2. world map painting tool, which can be used for making colored images from the images mentioned in part 1.</p>

<p><a href="http://www.outline-world-map.com/white-transparent-political-world-map-b5a"><img alt="Link to an example of a political world map with white land and transparent water areas - image" src="http://www.outline-world-map.com/map-images-tiny/tiny-political-white-transparent-thin-world-map-b5a.png" /></a>&nbsp;&nbsp; <a href="http://www.outline-world-map.com/blank-thick-white-world-map-b3c"><img alt="Link to an outline blank world map with white land and water areas - image" src="http://www.outline-world-map.com/map-images-tiny/tiny-blank-white-thick-world-map-b3c.png" /></a>&nbsp;&nbsp; <a href="http://www.outline-world-map.com/blank-gray-white-world-map-b9a"><img alt="An example blank world map with white land and gray water areas - image" src="http://www.outline-world-map.com/map-images-tiny/tiny-blank-thin-gray-white-world-map-b9a.png" /></a></p>

<h2><br />
Terms (definitions) on the website and source image</h2>

<p><br />
There are some phrases specified, as follows:<br />
By the phrase <strong>outline world map</strong> we mean, that the world maps on this website have all following common signs -<br />
the surfaces and borders can have quite approximate shapes instead of very precise shapes; the continents on the maps are surrounded with a (by default) black line.<br />
By using the combination of words <strong>blank world map</strong>, we are meaning, that there are no country borders on a such map.<br />
The words <strong>political world map</strong> are indicating on this website to an image, which has country borders (also black by default and with the same thickness as the outlines).<br />
<br />
Source: The outline world map images on this website were derived (graphically modified) from a colored map image, which is released to public domain (<a href="http://www.outline-world-map.com/links-to-other-web-sites#source">click here</a>).</p>
