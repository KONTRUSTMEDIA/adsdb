      <p>

        To avoid interruptions, charge your phone before calling so your call is not cut off.
      </p>

      <h5><span class="ez-toc-section" id="step-by-step-sassa-srd-status-check-via-phone-call"></span>Step-by-step: SASSA SRD status check via phone call<span class="ez-toc-section-end"></span></h5>
      <ol>
        <li>Have a pen and paper ready for notes and the <strong>call reference number</strong>.</li>
        <li>Prepare: <strong>ID number</strong>, <strong>cellphone number linked to SRD</strong>, and (if asked) your <strong>application details</strong>.</li>
        <li>Call <strong><a href="tel:+27800601011" data-wpel-link="internal">0800 60 10 11</a></strong> and follow the voice prompts.</li>
        <li>Select <strong>Option 1</strong> for <strong>SRD</strong> (menu wording may change over time; listen carefully).</li>
        <li>Select your language:
          <ul>
            <li><strong>English:</strong> press 1</li>
            <li><strong>Afrikaans:</strong> press 2</li>
            <li><strong>isiZulu:</strong> press 3</li>
            <li><strong>Sesotho:</strong> press 4</li>
            <li><strong>Xitsonga:</strong> press 5</li>
            <li><strong>Tshivenda:</strong> press 6</li>
            <li><strong>isiXhosa:</strong> press 7</li>
          </ul>
        </li>
        <li>When connected, ask the agent to confirm the <strong>status for the exact month</strong> you want to check.</li>
        <li>Write down the month result and the <strong>reference number</strong>, then match your wording under <a href="#srd-status-results" target="_self">SRD status results</a>.</li>
      </ol>

      <p class="notice">
        <strong>Safety tip:</strong> Do not share OTPs, bank PINs, or passwords. Only provide details when <strong>you</strong> dial the official number.
      </p>

        Take your ID and (if available) proof of the phone number used for SRD. Ask the official to confirm the <strong>exact month</strong> and <strong>exact status wording</strong>.
      </p>

  <section class="section alt-bg" id="appeals" aria-label="SASSA appeals for all grants">
    <h2><span class="ez-toc-section" id="sassa-appeals"></span>SASSA Appeals<span class="ez-toc-section-end"></span></h2>

    <p>
      A <strong>SASSA appeal</strong> is the formal process used to request a review of a grant decision when you believe the outcome is incorrect.
      Appeals follow a decision and ask for that decision to be reconsidered using the rules and evidence for the period being assessed.
    </p>

    <h3><span class="ez-toc-section" id="when-appeals-are-appropriate"></span>When appeals are appropriate<span class="ez-toc-section-end"></span></h3>
    <p>
      Appeals are not new applications and do not automatically speed up processing. Appeals work best when the underlying issue is fixed first
      (for example, identity mismatches or missing documents) and you submit supporting evidence that relates to the period being reviewed.
    </p>

    <p class="hero-note">
      Full internal overview (SRD + social grants):
      <a href="https://mystatus.co.za/sassa-srd-appeal/" target="_self" data-wpel-link="internal"><strong>SASSA appeals – full guide</strong></a>.
    </p>

  <section class="status-result" id="appeal-result-approved" aria-label="Appeal Approved meaning">
    <h3><span class="ez-toc-section" id="approved-appeal"></span>Approved (appeal)<span class="ez-toc-section-end"></span></h3>
    <p>
      “Approved” means your appeal succeeded for that month. Payment is typically initiated for the specific month you appealed,
      then follows SRD processing for that month (there isn’t one universal “appeal payday”).
    </p>
     <h4><span class="ez-toc-section" id="outcomes-and-payment-timing-if-your-appeal-status-is-%e2%80%9capproved%e2%80%9d"></span>Outcomes and payment timing (If your appeal status is &#8220;Approved&#8221;)<span class="ez-toc-section-end"></span></h4>
    <p>
      If an appeal is approved, payment is typically initiated for the specific month/period appealed. SRD does not have one fixed appeal payday;
      payment follows monthly processing for that month.
    </p>

  </section>

  <section class="status-result" id="appeal-result-declined" aria-label="Appeal Declined meaning">
    <h3><span class="ez-toc-section" id="declined-appeal"></span>Declined (appeal)<span class="ez-toc-section-end"></span></h3>
    <p>
      “Declined” means the appeal decision stays the same for the month that was reviewed on the portal.
      For that assessed month, the portal usually does not allow another appeal submission.
    </p>
 
    <div class="content-continue" role="note">
  <p>
    Learn more here:
    <a href="https://mystatus.co.za/sassa-status-check-declined/#appeal-declined" target="_self" data-wpel-link="internal">
      <strong>SRD Appeal Declined: What to do next</strong>
    </a>
  </p>
</div>

  <!-- FAQs (appeal status) -->
  <section class="section alt-bg news" id="appeal-status-meanings" aria-label="SRD Appeal Status FAQs">
    <h2><span class="ez-toc-section" id="srd-appeal-status-faqs"></span>SRD Appeal Status FAQs<span class="ez-toc-section-end"></span></h2>

    <details>
      <summary><strong>How long does an SRD appeal take?</strong></summary>
      <p>Appeal reviews can take several weeks, depending on volume and verification requirements.</p>
    </details>

    <details>
      <summary><strong>Can I appeal again if my appeal was declined?</strong></summary>
<p>
  No — if your appeal is declined for that assessed month, the portal does not allow another appeal for the same month/decision.
  If you still disagree after the final internal decision, use:
  <a href="https://mystatus.co.za/sassa-status-check-declined/#appeal-declined" target="_self" data-wpel-link="internal">
    <strong>SRD Appeal Declined: What to do next</strong>
  </a>.
</p>
    </details>

    <details>
      <summary><strong>Can I take a declined SRD appeal decision further?</strong></summary>
 <p>
  If you believe the appeal outcome was unlawful or procedurally unfair, you may challenge the decision through the courts, including the High Court.
  See:
  <a href="https://mystatus.co.za/sassa-status-check-declined/#appeal-declined" target="_self" data-wpel-link="internal">
    <strong>SRD Appeal Declined: What to do next</strong>
  </a>.
</p>


    </details>

    <details>
      <summary><strong>Does a declined appeal affect future SRD applications?</strong></summary>
      <p>No. Each month is assessed separately. You may qualify again in a future SRD assessment period if your circumstances change.</p>
    </details>
  </section>

    <h2><span class="ez-toc-section" id="sassa-identity-verification-why-it-matters-for-all-grants"></span>SASSA Identity Verification (Why It Matters for All Grants)<span class="ez-toc-section-end"></span></h2>

  <section class="section" id="eligibility" aria-label="Eligibility and application basics SASSA means test">
    <h2><span class="ez-toc-section" id="eligibility-application-basics-%e2%80%94-sassa-means-test"></span>Eligibility &amp; Application Basics — SASSA Means Test<span class="ez-toc-section-end"></span></h2>
    <p>
      A <strong>SASSA status check</strong> only shows meaningful results after you have <strong>applied</strong> for a grant.
      In practice, your ability to apply depends on whether you meet the <strong>SASSA means test</strong> and other grant-specific rules.
      The means test generally checks your <strong>income</strong> and, for some grants, your <strong>assets</strong> to determine eligibility.
    </p>

  <section class="section alt-bg" id="scams" aria-label="SASSA status check scam and fraud awareness">
    <h2><span class="ez-toc-section" id="sassa-status-check-scams-safety-tips"></span>SASSA Status Check Scams &amp; Safety Tips<span class="ez-toc-section-end"></span></h2>
    <ul>
      <li><strong>Check the domain:</strong> SRD status checks should be on <strong>srd.sassa.gov.za</strong>.</li>
      <li><strong>No fees:</strong> do not pay anyone to “unlock” your status, “release” a payment, or “approve” a grant.</li>
      <li><strong>No secret sharing:</strong> never share OTPs, banking PINs, or passwords.</li>
      <li><strong>Call-back scams:</strong> if someone calls claiming to be SASSA, end the call and dial the official number yourself.</li>
      <li><strong>Avoid random links:</strong> especially from comments and forwarded messages.</li>
    </ul>
    <p>
      Full internal guide:
      <a href="/avoid-srd-scams/" target="_self" data-wpel-link="internal">Avoid SRD scams &amp; how to spot fake SASSA websites</a>.

      var h = Math.round(header.getBoundingClientRect().height);
      if (h && h > 40 && h < 220) return h + 12;
    }
    return fallback;
  }

  function scrollToId(id, behavior) {
    var el = byId(id);
    if (!el) return;
    var y = window.pageYOffset + el.getBoundingClientRect().top - getStickyOffset();
    window.scrollTo({ top: Math.max(0, y), behavior: behavior || 'smooth' });
  }

  function bindGrantSelector() {
    var sel = byId('grant-selector');
    if (!sel) return;
    sel.addEventListener('change', function () {
      var v = sel.value;
      if (!v) return;
      try { history.replaceState(null, '', '#' + v); } catch (e) {}
      scrollToId(v, 'smooth');
    });
  }
function bindDeclineReasonSelector() {
  var sel = byId('decline-reason-selector');
  if (!sel) return;

  sel.addEventListener('change', function () {
    var url = sel.value;
    if (!url) return;
    window.location.href = url;
  });
}

  function initHashLanding() {
    if (!window.location.hash || window.location.hash.length < 2) return;
    var id = window.location.hash.slice(1);

    // run twice to beat async header/layout shifts
    window.setTimeout(function () { scrollToId(id, 'auto'); }, 60);
    window.setTimeout(function () { scrollToId(id, 'auto'); }, 650);

    var sel = byId('grant-selector');
    if (sel) {
      for (var i = 0; i < sel.options.length; i++) {
        if (sel.options[i].value === id) { sel.value = id; break; }
      }
    }
  }

  // ===== Loader (toast only when popups are blocked) =====
  function getLoader() {
    return byId('official-portal-loader');
  }

function hideLoader() {
  var loader = getLoader();
  if (!loader) return;
  loader.hidden = true;
  loader.style.display = 'none';
}

function showLoaderToast() {
  var loader = getLoader();
  if (!loader) return;

  loader.hidden = false;
  loader.style.display = 'block';

  loader.style.position = 'fixed';
  loader.style.left = '12px';
  loader.style.right = '12px';
  loader.style.bottom = '14px';
  loader.style.maxWidth = '640px';
  loader.style.margin = '0 auto';
  loader.style.zIndex = '99999';
}
  function hideLoaderSoon() {
    window.setTimeout(function () { hideLoader(); }, 1400);
  }

  // ===== Popup opener =====
  function popupFeatures() {
    var wWidth  = Math.min(980, Math.floor(window.screen.availWidth * 0.9));
    var wHeight = Math.min(720, Math.floor(window.screen.availHeight * 0.85));
    var left = Math.max(0, Math.floor((window.screen.availWidth  - wWidth)  / 2));
    var top  = Math.max(0, Math.floor((window.screen.availHeight - wHeight) / 2));

    return [
      'popup=yes',
      'resizable=yes',
      'scrollbars=yes',
      'width=' + wWidth,
      'height=' + wHeight,
      'left=' + left,
      'top=' + top
    ].join(',');
  }

  function openOfficial(url, name) {
    if (!url) return;

    // MUST be direct (no timeouts) or iOS/Safari blocks it
    var win = window.open(url, name || 'sassaOfficial', popupFeatures());

    if (win) {
      try { win.opener = null; } catch (e) {}
      try { win.focus(); } catch (e) {}
      return;
    }

    // Popup blocked → show toast + fall back
    showLoaderToast();

    var win2 = window.open(url, '_blank', 'noopener,noreferrer');
    if (!win2) {
      // last resort (still works, but navigates away)
      window.location.href = url;
    }

    hideLoaderSoon();
  }

  // ===== Bindings =====
  function bindOfficialLinks() {
    var links = document.querySelectorAll('a.open-official');
    if (!links || !links.length) return;

    for (var i = 0; i < links.length; i++) {
      (function (a) {
        a.addEventListener('click', function (e) {
          // allow ctrl/cmd click to behave normally
          if (e.ctrlKey || e.metaKey || e.shiftKey || e.altKey) return;

          try { e.preventDefault(); } catch (err) {}
          var url = a.getAttribute('href');
          var name = a.getAttribute('data-win') || 'sassaOfficial';
          openOfficial(url, name);
        });
      })(links[i]);
    }
  }

  function bindButtons() {
    var srdBtn = byId('open-srd-status');
    var servicesBtn = byId('open-services-portal');

    if (srdBtn) {
      srdBtn.addEventListener('click', function () {
        openOfficial('https://srd.sassa.gov.za/sc19/status', 'sassaSrdStatus');
      });
    }

    if (servicesBtn) {
      servicesBtn.addEventListener('click', function () {
        openOfficial('https://services.sassa.gov.za', 'sassaServicesPortal');
      });
    }
  }

  function bind() {
    hideLoader();          // keep loader hidden on page load
    bindButtons();         // hero buttons
    bindOfficialLinks();   // any .open-official links (including appeal button)
    bindGrantSelector();   // payment dates selector
    bindDeclineReasonSelector(); // Decline Reason Page Selector selector
    initHashLanding();     // hash scrolling
  }

  if (document.readyState === 'loading') {
    document.addEventListener('DOMContentLoaded', bind);
  } else {
    bind();
  }
})();
