	<div class="col-md-5">
		<p><h2>Pin Code (Postal Index Number) Structure:</h2></p>
		<p>The PIN Code (Postal Index Number) was <b>introduced on 15 August 1972</b>. It is an <b>alphanumeric 6 digits code for the post office numbering</b> or post code system used by Indian Post. </p>
		<br>
		<p><b>First digit</b> of the PIN code indicates the <b>region</b>, <b>second digit</b> indicates the <b>sub-region</b>, <b>third digit</b> indicates the <b>district within the region</b>. The <b>final three digits</b> are assigned to <b>individual post offices</b>.</p>
	</div>
	
	<div class="col-md-12 newimagecls">
		<p><h2>How Pin Code Finder Works?</h2></p>
		<p>It is very easy to get the Pin Code by using the Pin Code Finder, Suppose you are looking for the <b>Pin Code of the Post Office of Dalippur Pratapgarh, District: Pratapgarh, State: Uttar Pradesh</b>. You just need to Select the <b>State: Uttar Pradesh</b> from the State Dropdown and then Select <b>District: Pratapgarh</b> from District Dropdown and now select <b>Postoffice: Dalippur Pratapgarh</b> from Postoffice Dropdown and you will get the Pin Code below. If you have already the Pin Code you can get the information by using the search from the <a target="_blank" href="https://www.searchpincode.com/search-by-pin-code"><b>Search By Pin Code.</b></a></p>
