    <h2>
      <a href="/docs/linux/linux-about/">Linux版本使用说明</a>
    </h2>
    
  <h5>July 12, 2023</h5>



  

  




    <p><h3 id="重要">
  重要
  <a class="anchor" href="#%e9%87%8d%e8%a6%81">#</a>
</h3>
<p><span style="color:red">任意一个Linux版本随时可能会被服务器ban掉, 所以如果发现无法使用, 请更新到最新版本</span></p>
<h3 id="设计">
  设计
  <a class="anchor" href="#%e8%ae%be%e8%ae%a1">#</a>
</h3>
<p>快贴Linux版本采用BSD经典的daemon + control的设计方式. 支持Linux多用户系统.</br>
Linux版本依然采用弱化离、在线状态的模式, 即使不登录与无网络的情况, 快贴依然可以正常工作.</p>
<h3 id="安全">
  安全
  <a class="anchor" href="#%e5%ae%89%e5%85%a8">#</a>
</h3>
<p>快贴daemon使用unix socket通信方式, 不监听端口. </br>
同时socket文件权限为0600, 仅用户可读写, 其他人无法读写数据.</br></p>
<h3 id="开源">
  开源
  <a class="anchor" href="#%e5%bc%80%e6%ba%90">#</a>
</h3>
<p>快贴从法律上没有开源的义务, 快贴不引用任何带有强制开源许可证的第三方源码. 可使用 <code>clipctl about credits</code> 查看引用的第三方内容与license.</p>
<h3 id="文件">
  文件
  <a class="anchor" href="#%e6%96%87%e4%bb%b6">#</a>
</h3>
<p>安装文件包括:</p>
<div class="highlight"><pre tabindex="0" style="color:#f8f8f2;background-color:#272822;-moz-tab-size:4;-o-tab-size:4;tab-size:4;"><code class="language-console" data-lang="console"><span style="display:flex;"><span>clipd # 负责与快贴服务器接口通信的daemon. 
</span></span></code></pre></div><div class="highlight"><pre tabindex="0" style="color:#f8f8f2;background-color:#272822;-moz-tab-size:4;-o-tab-size:4;tab-size:4;"><code class="language-console" data-lang="console"><span style="display:flex;"><span>clipctl # 用户与clipd通信的接口cli. 
</span></span></code></pre></div><h3 id="功能">
  功能
  <a class="anchor" href="#%e5%8a%9f%e8%83%bd">#</a>
</h3>
<p>目前Linux版本支持文本与图片的同步. 另外包括一些功能并未开放. 具体详见下面的使用方法.</p>
<h3 id="使用方法">
  使用方法
  <a class="anchor" href="#%e4%bd%bf%e7%94%a8%e6%96%b9%e6%b3%95">#</a>

    <h2>
      <a href="/docs/linux/linux-trick/">Linux快贴能做什么?</a>
    </h2>
    
  <h5>July 10, 2023</h5>



  

  




    <p><h3 id="重要">
  重要
  <a class="anchor" href="#%e9%87%8d%e8%a6%81">#</a>
</h3>
<p><span style="color:red">任意一个Linux版本随时可能会被服务器ban掉, 所以如果发现无法使用, 请更新到最新版本</span></p>
<h3 id="当前linux版本能做什么">
  当前Linux版本能做什么?
  <a class="anchor" href="#%e5%bd%93%e5%89%8dlinux%e7%89%88%e6%9c%ac%e8%83%bd%e5%81%9a%e4%bb%80%e4%b9%88">#</a>
</h3>
<blockquote>
<p>1 备用的剪贴板历史记录存储</p></blockquote>
<p>快贴采用local-first的设计方案, 简单说剪贴板数据主要储存在本地, 通过服务端实时同步其他端的剪贴板数据. 如果某一个端未在线就无法实时同步数据, 这时候就可以使用Linux版本来弥补这一缺陷.</p>
<p>比如配合会员的历史记录与搜索功能:</p>
<p>host:</p>
<div class="highlight"><pre tabindex="0" style="color:#f8f8f2;background-color:#272822;-moz-tab-size:4;-o-tab-size:4;tab-size:4;"><code class="language-console" data-lang="console"><span style="display:flex;"><span>ssh user@server clipctl search asdf 
</span></span></code></pre></div><p>docker:</p>
<div class="highlight"><pre tabindex="0" style="color:#f8f8f2;background-color:#272822;-moz-tab-size:4;-o-tab-size:4;tab-size:4;"><code class="language-console" data-lang="console"><span style="display:flex;"><span>ssh user@server docker exec -i clipd clipctl search asdf 
</span></span></code></pre></div><p>找到需要的剪贴板历史后:</p>
<p>host:</p>
<div class="highlight"><pre tabindex="0" style="color:#f8f8f2;background-color:#272822;-moz-tab-size:4;-o-tab-size:4;tab-size:4;"><code class="language-console" data-lang="console"><span style="display:flex;"><span>ssh user@server clipctl set touch &#39;clip-id&#39;
</span></span></code></pre></div><p>docker:</p>
<div class="highlight"><pre tabindex="0" style="color:#f8f8f2;background-color:#272822;-moz-tab-size:4;-o-tab-size:4;tab-size:4;"><code class="language-console" data-lang="console"><span style="display:flex;"><span>ssh user@server docker exec -i clipd clipctl set touch &#39;clip-id&#39;
</span></span></code></pre></div><blockquote>
<p>2 发送剪贴板数据到远程Linux版本, 并同步到其他端</p></blockquote>
<p>host:</p>
<div class="highlight"><pre tabindex="0" style="color:#f8f8f2;background-color:#272822;-moz-tab-size:4;-o-tab-size:4;tab-size:4;"><code class="language-console" data-lang="console"><span style="display:flex;"><span>echo &#39;asdf&#39; | ssh user@server clipctl add text - 
</span></span></code></pre></div><p>docker:</p>
<div class="highlight"><pre tabindex="0" style="color:#f8f8f2;background-color:#272822;-moz-tab-size:4;-o-tab-size:4;tab-size:4;"><code class="language-console" data-lang="console"><span style="display:flex;"><span>echo &#39;asdf&#39; | ssh user@server docker exec -i clipd clipctl add text -
</span></span></code></pre></div><blockquote>
<p>3 自己制作网页版</p>
        <a href="/docs/linux/linux-trick/">...</a>

      <a href="/posts/new-sign/">Mac版本更新签名</a>
    </h2>
    
  <h5>July 28, 2020</h5>



  

  




    <p><p>Mac版本的新包更新了签名证书, 因此有如下改动:</p>
<ol>
<li>LOGO 变化.</li>
</ol>
<p>
  <img src="/newlogo.png" alt="logo changed" /></p>
<ol start="2">
<li>
<p>需要重新登录.</br>
签名变化后, 读取之前的加密时系统会弹出访问快贴前一版本存储的keychain授权的对话框. </br>
考虑到有小部分用户不理解这个对话框的含义, 因此新版直接使用新的地址来存储加密数据. </br></p>
</li>
<li>
<p>需要把老版本删除.</br>
新包不会覆盖老版本.</p>
</li>
</ol>
    </p>

    <h2>
      <a href="/posts/faq/">FAQ</a>
    </h2>
    
  <h5>May 10, 2020</h5>



  

  




    <p><h3 id="q-快贴是做什么的">
  Q. 快贴是做什么的?
  <a class="anchor" href="#q-%e5%bf%ab%e8%b4%b4%e6%98%af%e5%81%9a%e4%bb%80%e4%b9%88%e7%9a%84">#</a>
</h3>
<p>A. 简单说, 快贴是一款剪贴板工具, 设计的目的是增加程序开发人员的工作效率, 主要场景是: 存储剪贴板历史记录, 使用插件
  <a href="https://github.com/cissusnar/cpluginscript/tree/master/cppclass.cplugin">批量生成代码</a>. 同时也可以帮助开发人员多平台的同步剪贴板.</p>
<h3 id="q-快贴适合普通用户使用吗">
  Q. 快贴适合普通用户使用吗?
  <a class="anchor" href="#q-%e5%bf%ab%e8%b4%b4%e9%80%82%e5%90%88%e6%99%ae%e9%80%9a%e7%94%a8%e6%88%b7%e4%bd%bf%e7%94%a8%e5%90%97">#</a>
</h3>
<p>A. <span style="color:red">不推荐</span>普通用户使用, 快贴适合有
  <a href="https://www.lua.org/">脚本语言</a>基础或比较geek的用户使用快贴.</p>
<h3 id="q-需要联网登录才能使用吗">
  Q. 需要联网、登录才能使用吗?
  <a class="anchor" href="#q-%e9%9c%80%e8%a6%81%e8%81%94%e7%bd%91%e7%99%bb%e5%bd%95%e6%89%8d%e8%83%bd%e4%bd%bf%e7%94%a8%e5%90%97">#</a>
</h3>
<p>A. 不需要. 快贴任何一款客户端都可以独立使用, 无需登录, 即使登录后大部分功能都不区分离、在线状态.</p>
<h3 id="q-快贴稳定吗">
  Q. 快贴稳定吗?
  <a class="anchor" href="#q-%e5%bf%ab%e8%b4%b4%e7%a8%b3%e5%ae%9a%e5%90%97">#</a>
</h3>
<p>A. 快贴是稳定版本的软件,快贴Mac的第一个版本在2014年12月发布, 随后iOS发布, 2015年3月快贴发布了iOS、Mac、Android和Windows四个平台的版本. 快贴是国内第一款跨平台的剪贴板软件.</p>
<h3 id="q-快贴可以同步哪些数据">
  Q. 快贴可以同步哪些数据?
  <a class="anchor" href="#q-%e5%bf%ab%e8%b4%b4%e5%8f%af%e4%bb%a5%e5%90%8c%e6%ad%a5%e5%93%aa%e4%ba%9b%e6%95%b0%e6%8d%ae">#</a>
</h3>
<p>A. 快贴可以同步文本、图片和文件, 文件目前仅支持电脑端向其他平台同步.</p>
<h3 id="q-为什么复制文件后不能同步">
  Q. 为什么复制文件后不能同步?
  <a class="anchor" href="#q-%e4%b8%ba%e4%bb%80%e4%b9%88%e5%a4%8d%e5%88%b6%e6%96%87%e4%bb%b6%e5%90%8e%e4%b8%8d%e8%83%bd%e5%90%8c%e6%ad%a5">#</a>
</h3>
<p>A. 目前最大支持5MB大小的文件, 另外文件默认设置为隐私数据, 需要手动点击‘解除隐私’才可以同步. 但您可以机密设置里面取消手动同步.</p>
<h3 id="q-为什么不能排除密码管理软件">
  Q. 为什么不能排除密码管理软件?
  <a class="anchor" href="#q-%e4%b8%ba%e4%bb%80%e4%b9%88%e4%b8%8d%e8%83%bd%e6%8e%92%e9%99%a4%e5%af%86%e7%a0%81%e7%ae%a1%e7%90%86%e8%bd%af%e4%bb%b6">#</a>
</h3>
<p>A. 通过剪贴板的管理密码是极其
  <a href="https://security.stackexchange.com/questions/33428/is-a-password-in-the-clipboard-vulnerable-to-attacks">愚蠢的设计</a>, 密码管理软件的锅剪贴板管理软件不背. 请使用Chrome自带的密码管理器, 或等待操作系统增加新的接口.</p>
<h3 id="q-电脑版打开没有反应">
  Q. 电脑版打开没有反应?
  <a class="anchor" href="#q-%e7%94%b5%e8%84%91%e7%89%88%e6%89%93%e5%bc%80%e6%b2%a1%e6%9c%89%e5%8f%8d%e5%ba%94">#</a>
</h3>
<p>A. Mac和Windows快贴运行后会出现在任务栏, 可以点击任务栏的快贴图标或者使用快捷键: shift + command(win) + v 来呼出快贴菜单.</p>
