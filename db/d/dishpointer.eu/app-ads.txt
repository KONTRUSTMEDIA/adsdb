function trackPosition() {
  if(posId == -1) { smoothScrollTo("#mapdiv")
    if (navigator.geolocation) {
      posId = navigator.geolocation.watchPosition(positionSuccess, positionError, positionOptions);
      customControlDiv.firstChild.title = 'Click to stop tracking your location';
      customControlDiv.firstChild.firstChild.innerHTML = `Stop tracking`;
      setId = setInterval(function(){blink()}, 1000);
    } else {
      positionError(-1);
    }
  } else {
  navigator.geolocation.clearWatch(posId);
  posId = -1;
  accuracyCircle.setMap(null);
  customControlDiv.firstChild.title = 'Wifi and/or GPS needed for best results';
  customControlDiv.firstChild.firstChild.innerHTML = `Track my location`;
  clearInterval(setId);
  }
}


  function positionSuccess(position) {
    /* Centre the map on the new location */
    var coords = position.coords || position.coordinate || position;
    var newLat = RoundNum2(coords.latitude, 5);
    var newLng = RoundNum2(coords.longitude, 5);
    if(newLat != document.getElementById('Latitude').value || newLng != document.getElementById('Longitude').value) {
    document.getElementById('Latitude').value = newLat;
    document.getElementById('Longitude').value = newLng;
    DoCalc1();
    document.getElementById('map_canvas').style.display='block';
    google.maps.event.trigger(map, 'resize');
    }
    /* Add the circle to the map. */
    accuracyCircle.setMap(map);
    accuracyCircle.setCenter(new google.maps.LatLng(newLat, newLng));
    accuracyCircle.setRadius(coords.accuracy);
    map.fitBounds(accuracyCircle.getBounds());
}

function positionError(err) {
    if (navigator.geolocation) {
      navigator.geolocation.getCurrentPosition(positionSuccess, positionErrorNoGPS, positionOptionsNoGPS);
    } else {
      positionErrorNoGPS(-1);
    }

}

Communications satellites and weather satellites are often placed in geostationary orbits, so that the satellite antennas (located on Earth) that communicate with them do not have to rotate to track them.<br /><a class="w3-btn w3-pale-blue w3-border-0 w3-padding-0" style="display: inline" href="https://en.wikipedia.org/wiki/Geostationary_orbit" target="_blank">Wikipedia</a></p>

<h4>
  <button onclick="w3ShowHide('skewangle')" class="w3-btn-block w3-left-align w3-dark-grey">
                <em>LNB skew</em> >
</button>
</h4>
 <div id="skewangle" class="w3-accordion-content w3-container w3-show">
<br />
                Most satellite TV transmissions use the so called linear polarisation, which in practice can be either vertical or horizontal. This means that the vector of the electric field of the electromagnetic wave received from the satellite is in a horizontal plane for some frequencies, and in a vertical plane for others. For optimal reception, the receiving antenna should be parallel to the satellite's emitting antenna. This will also ensure that the signals from the other polarity are completely eliminated, since they can be on a nearby frequency or even the same frequency. This possibility of separating signals by polarity practically allows doubling the number of transmissions on a satellite.
<div class="w3-row-padding">

<div class="w3-twothird w3-padding-medium">
                <p>&nbsp;</p>But, because satellites are on a circular orbit around the Equator, the vertical (and horizontal) of the satellite are rotated according to the vertical and horizontal at the orbital position of the satellite. For an observer on Earth, only the satellite at the local meridian will have the same vertical and horizontal (for example in London this is approximately the satellite Thor at 0.8&deg;W, for all the other satellites the vertical and horizontal will be tilted, the more so for satellites that are farther to the East, or to the West.</div>

<div class="w3-third">
<div class="w3-card-12 w3-margin">
                <p>&nbsp;</p><img class='lazyload' data-src="/images/polarisation.jpg" alt="polarisation" style="width:100%">
<div class="w3-container w3-center">
                <p><em>Horizontal and vertical polarisation.</em></p>
</div>
</div>
</div>
</div>

<div class="w3-row-padding">
<div class="w3-third">
<div class="w3-card-12 w3-margin">
                <p><img class='lazyload' data-src="/images/lnb_hv_rods.jpg" alt="lnb rods" style="width:100%"></p>
<div class="w3-container w3-center">
                <p><em>The receiving antenna is in fact located inside the LNB<br>Actually there are 2 antennas, one for horizontal polarisation, the other one for vertical polarisation.</em></p>
</div>
</div>
</div>
<div class="w3-twothird w3-padding-medium">
<br /><br /><br /><br />
<div class="w3-container w3-pale-blue w3-leftbar w3-border-blue w3-padding-small">
                <p>The LNB skew is the angle of the LNB from its vertical position, that ensures the receiving antennas inside the LNB are parallel to the emitting antennas of the satellite, considering all the above.</p>
</div>
</div>
</div>
                <br><em>The following information is for the Northern Hemishpere.</em>

<div class="w3-row-padding">
<div class="w3-quarter">

<div class="w3-container w3-center">
                <p><em>For satellites that are to the West relative to the dish, the skew will be shown as positive, and the LNB should be turned counterclockwise, if facing the dish from the front.</em></p>
</div>
</div>
</div>

<div class="w3-quarter">
<div class="w3-card-12 w3-margin">
                <p><img class='lazyload' data-src="/images/sqewneg.jpg" alt="negative skew" style="width:100%"></p>

</div>

                <p>&nbsp;</p>This is only needed for signals with linear polarisation. For circular polarisation, there is no need to adjust the LNB skew.                <p>&nbsp;</p>
                <p>For a motorised dish, as the dish moves on its curve in order to receive the satellites, the LNB skew changes for the optimum reception of each satellite.</p>
