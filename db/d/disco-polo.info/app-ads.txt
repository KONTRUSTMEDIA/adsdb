<h1 class="text-muted h5">Disco-polo.info – portal disco polo i muzyki disco polo</h1>

<p class="font6 lineh-1 text-muted text-justify pr-2">
Chcesz być na bieżąco z tym, co dzieje się na scenie disco polo? 
<strong><a href="https://disco-polo.info/" class="text-muted">Disco-polo.info</a></strong> 
to największy <strong>portal disco polo</strong>, na którym codziennie publikujemy 
<strong>najnowsze newsy disco polo</strong>, aktualności ze świata artystów oraz 
<strong>nowości disco polo</strong>. Informujemy o premierach piosenek, zapowiedziach koncertów 
i najważniejszych wydarzeniach związanych z muzyką disco polo w Polsce.
</p>

<p class="font6 lineh-1 text-muted text-justify pr-2">
Na portalu znajdziesz codziennie aktualizowane 
<strong><a href="https://disco-polo.info/artykuly" class="text-muted">artykuły disco polo</a></strong>, 
w tym <strong><a href="https://disco-polo.info/artykuly/premiera" class="text-muted">premiery disco polo</a></strong>, 
<strong><a href="https://disco-polo.info/artykuly/koncert" class="text-muted">koncerty disco polo</a></strong> 
oraz sprawdzone informacje z życia <strong>gwiazd disco polo</strong>. 
Regularnie odwiedzając nasz serwis, masz pewność, że nie ominie Cię żadna nowość.
</p>

<h2 class="text-muted h5">Piosenki disco polo i teledyski disco polo</h2>

<p class="font6 lineh-1 text-muted text-justify pr-2">
<strong>Muzyka disco polo</strong> to taneczne rytmy, chwytliwe melodie i teksty, które szybko wpadają w ucho. 
Na disco-polo.info możesz legalnie słuchać 
<strong><a href="https://disco-polo.info/legalne-mp3" class="text-muted">piosenek disco polo online</a></strong> 
oraz oglądać <strong><a href="https://disco-polo.info/teledyski" class="text-muted">najnowsze teledyski disco polo</a></strong>. 
Prezentujemy zarówno klasyczne hity, jak i zupełnie nowe produkcje, które dopiero podbijają listy przebojów.
</p>

<p class="font6 lineh-1 text-muted text-justify pr-2">
Jeśli szukasz sprawdzonych zestawień, sprawdź nasze 
<strong><a href="https://disco-polo.info/top20" class="text-muted">TOP 20 – najlepsze teledyski disco polo</a></strong>, 
gdzie regularnie publikujemy najpopularniejsze klipy wybierane przez fanów.
</p>

<h2 class="text-muted h5">Składanki disco polo i hity, które łączą pokolenia</h2>

<p class="font6 lineh-1 text-muted text-justify pr-2">
<strong>Hity disco polo</strong> od lat goszczą na weselach, imprezach rodzinnych i festiwalach muzycznych. 
To gatunek, który łączy pokolenia i potrafi porwać do zabawy każdego – bez względu na wiek. 
Na naszym portalu regularnie prezentujemy najpopularniejsze utwory oraz składanki disco polo, 
które idealnie sprawdzają się na każdą okazję.
</p>

<p class="font6 lineh-1 text-muted text-justify pr-2">
Disco-polo.info to miejsce stworzone dla fanów muzyki tanecznej. 
Publikujemy każdy istotny <strong>news disco polo</strong>, pokazujemy nowe zespoły na scenie 
i udostępniamy najlepsze <strong>teledyski disco polo</strong> w pełni legalnie. 
Odwiedzaj nas codziennie i bądź zawsze na bieżąco z tym, co najważniejsze w świecie disco polo.
</p>

// INP OPTIMIZER - YIELD TO MAIN THREAD

(function() {
  // Debounce dla scroll eventów
  function debounce(func, wait) {
    var timeout;
    return function() {
      var context = this, args = arguments;
      clearTimeout(timeout);
      timeout = setTimeout(function() {
        func.apply(context, args);
      }, wait);
    };
  }
  // Throttle dla częstych eventów
  function throttle(func, limit) {
    var inThrottle;
    return function() {
      var args = arguments;
      var context = this;
      if (!inThrottle) {
        func.apply(context, args);
        inThrottle = true;
        setTimeout(function() { inThrottle = false; }, limit);
      }
    };
  }
  // Globalne helpery
  window.dpDebounce = debounce;
  window.dpThrottle = throttle;
})();

(function() {
  var taskQueue = [];
  var isProcessing = false;

  window.scheduleTask = function(task, priority) {
    taskQueue.push({ task: task, priority: priority || 0 });
    if (!isProcessing) processQueue();
  };

  function processQueue() {
    if (taskQueue.length === 0) {
      isProcessing = false;
      return;
    }

    isProcessing = true;
    
    // Sortuj po priorytecie
    taskQueue.sort(function(a, b) { return b.priority - a.priority; });
    
    var item = taskQueue.shift();
    
    requestIdleCallback(function(deadline) {
      try {
        if (deadline.timeRemaining() > 0) {
          item.task();
        } else {
          // Odłóż na później jeśli brak czasu
          taskQueue.unshift(item);
        }
      } catch(e) {
        console.error('Task error:', e);
      }
      processQueue();
    }, { timeout: 1000 });
  }

  // dpDefer - uruchamiaj cięższe rzeczy dopiero gdy przeglądarka ma chwilę
  window.dpDefer = function (fn, timeout) {
    if ('requestIdleCallback' in window) {
      requestIdleCallback(fn, { timeout: timeout || 1500 });
    } else {
      setTimeout(fn, 1);
    }
  };

  // dpYield - prosty "yield" żeby nie blokować INP
  window.dpYield = function () {
    return new Promise(function (r) { setTimeout(r, 0); });
  };
})();

(function() {
  if (typeof jQuery !== 'undefined') return; // Already loaded
  
  var jqLoaded = false;
  var bsLoaded = false;
  
  function loadJQuery() {
    if (jqLoaded) return;
    jqLoaded = true;
    
    var jq = document.createElement('script');
    jq.src = 'https://code.jquery.com/jquery-3.7.1.min.js';
    jq.onload = function() {
      loadBootstrap();
    };
    document.body.appendChild(jq);
  }
  
  function loadBootstrap() {
    // Guard: sprawdź czy Bootstrap już nie istnieje
    if (bsLoaded || (typeof jQuery !== 'undefined' && typeof jQuery.fn.modal !== 'undefined')) {
      return;
    }
    
    bsLoaded = true;

    bs.src = 'https://cdn.jsdelivr.net/npm/bootstrap@4.6.2/dist/js/bootstrap.bundle.min.js';

      var bsLocal = document.createElement('script');
      bsLocal.src = '/js/bootstrap.min.js';
      document.body.appendChild(bsLocal);
    };
    document.body.appendChild(bs);
  }
  
  var events = ['mousedown', 'touchstart', 'keydown', 'scroll'];
  events.forEach(function(e) {
    window.addEventListener(e, loadJQuery, {once: true, passive: true});
  });
  
  setTimeout(loadJQuery, 5000);
})();
