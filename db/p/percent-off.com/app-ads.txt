<script>
// Desktop dropdown toggle
document.addEventListener('DOMContentLoaded', function() {
  const button = document.getElementById('calcDropdown');
  const dropdown = document.getElementById('dropdownMenu');

  if (button && dropdown) {
    button.addEventListener('click', function(e) {
      e.stopPropagation();
      dropdown.classList.toggle('show');
      const isOpen = dropdown.classList.contains('show');
      button.setAttribute('aria-expanded', isOpen ? 'true' : 'false');
    });
  }
});

function toggleMobileMenu() {
  const nav = document.getElementById('mainNav');
  const button = nav.querySelector('.mobile-toggle');
  const isOpen = nav.classList.contains('mobile-open');

  if (isOpen) {
    nav.classList.remove('mobile-open');
    button.setAttribute('aria-expanded', 'false');
  } else {
    nav.classList.add('mobile-open');
    button.setAttribute('aria-expanded', 'true');
  }
}

function toggleTheme() {
  document.body.classList.toggle('dark-theme');
  const isDark = document.body.classList.contains('dark-theme');
  localStorage.setItem('theme', isDark ? 'dark' : 'light');
}

// Close dropdowns when clicking outside
document.addEventListener('click', function(event) {
  const dropdown = document.getElementById('dropdownMenu');
  const button = document.getElementById('calcDropdown');
  const nav = document.getElementById('mainNav');
  const mobileButton = nav?.querySelector('.mobile-toggle');

  // Close desktop dropdown
  if (dropdown && button && !button.contains(event.target) && !dropdown.contains(event.target)) {
    dropdown.classList.remove('show');
    button.setAttribute('aria-expanded', 'false');
  }

  // Close mobile menu
  if (nav && mobileButton && !nav.contains(event.target)) {
    nav.classList.remove('mobile-open');
    mobileButton.setAttribute('aria-expanded', 'false');
  }
});

// Load theme preference
document.addEventListener('DOMContentLoaded', function() {
  const theme = localStorage.getItem('theme');
  if (theme === 'dark') {
    document.body.classList.add('dark-theme');
  }
});

            <section id="description">
                <h2>What is a Percent-Off Calculator?</h2>
                <div>
                    <p>A percent-off calculator is a practical tool that instantly computes how much you'll save and pay when an item is discounted by a certain percentage. Whether you're shopping during Black Friday sales, using a coupon code, or comparing prices across stores, this calculator eliminates the mental math and ensures you understand the true cost before making a purchase.</p>

                    <p>The calculator uses a simple two-step formula: First, it multiplies the original price by the discount percentage and divides by 100 to find the amount saved. Second, it subtracts that savings from the original price to determine your final cost. For example, a 25% discount on a $200 item means you save $50 and pay $150.</p>

                    <p>Beyond simple calculations, understanding percent-off discounts helps you make smarter purchasing decisions. A 30% discount on a $500 laptop ($350 final price) might be a better deal than a $100 gift card promotion, depending on your needs. This calculator provides instant clarity so you can evaluate offers quickly and confidently.</p>
                </div>
            </section>

            <section id="scenarios">
                <h2>Common Discount Scenarios Explained</h2>

                <h3>Stacking Discounts: Percent-Off Plus Coupon</h3>
                <p>When you have both a percentage discount and a dollar-amount coupon, the order matters. Most retailers apply the percent-off discount first, then subtract the coupon from that discounted price. For example:</p>
                <ul class="bulletul">
                    <li class="bullet">Item: $100 with 20% off = $80</li>
                    <li class="bullet">Then apply $10 coupon: $80 - $10 = <strong>$70 final price</strong></li>
                    <li class="bullet">You save $30 total (not $100 - $70 = $30, confirming the calculation)</li>
                </ul>
                <p>Some stores don't allow stacking at all, so always read the fine print on coupons and promotional codes.</p>

                <h3>Member-Exclusive Discounts</h3>
                <p>Loyalty programs, student discounts, and military discounts typically range from 10-20% off. These modest discounts are valuable when used consistently on regular purchases. If you shop at a store monthly, a 15% member discount can add up to hundreds of dollars in annual savings.</p>

                <h3>Price Matching with Percentage Discounts</h3>
                <p>Many retailers offer price matching, but when competitors advertise percentage discounts, you need to calculate the exact final price. For instance, Store A selling an item for $120 with no discount vs. Store B selling it for $150 with 25% off ($112.50 final price) – Store B is actually cheaper despite the higher original price.</p>

                <h3>Clearance Markdowns: 30%, 50%, 70% Off</h3>
                <p>Clearance items typically go through multiple markdown cycles. An item might start at 30% off, then increase to 50% off after two weeks, and finally 70% off before being removed. If the item isn't urgent, waiting for deeper discounts can triple your savings, though popular sizes and styles may sell out.</p>
            </section>

                <h2>Smart Shopping Tips to Maximize Savings</h2>

                <div>
                    <meta content="1">
                    <h3>1. Know the True Value of Discounts</h3>
                    <p>Not all percentage discounts are created equal. A 50% discount on an overpriced item might still cost more than a 20% discount on a fairly priced competitor's product. Always research the market price for items before being swayed by large percentage discounts.</p>
                </div>

                <div>
                    <meta content="2">
                    <h3>2. Combine Discounts with Cashback and Rewards</h3>
                    <p>Layer your savings by using cashback credit cards (1-5% back), shopping portal bonuses (2-10% back), and retailer loyalty points on top of the advertised discount. A 25% sale combined with 5% total cashback effectively becomes a 28.75% total discount.</p>
                </div>

                <div>
                    <meta content="3">
                    <h3>3. Calculate the Break-Even Point</h3>
                    <p>For membership programs that offer discounts (like Amazon Prime or Costco), calculate how much you'd need to spend annually to make the membership fee worthwhile. If membership costs $100/year and offers 10% off all purchases, you'd need to spend $1,000 annually to break even.</p>
                </div>

                <div>
                    <meta content="4">
                    <h3>4. Beware of Inflated "Original Prices"</h3>
                    <p>Some retailers inflate the "original price" to make discounts appear larger. A shirt marked "$100, now 60% off!" ($40 final price) might never have sold at $100. Use price tracking tools and your knowledge of typical market prices to spot genuine deals.</p>
                </div>

                <div>
                    <meta content="5">
                    <h3>5. Time Your Purchases for Maximum Discounts</h3>
                    <div>
                        <p>Different products have predictable discount cycles:</p>
                        <ul class="bulletul">
                            <li class="bullet"><strong>Electronics:</strong> Best deals during Black Friday, Cyber Monday, and new model releases</li>
                            <li class="bullet"><strong>Clothing:</strong> End of season (January for winter, July for summer) brings 40-70% off</li>
                            <li class="bullet"><strong>Furniture:</strong> Presidents Day, Memorial Day, and Labor Day feature major sales</li>
                            <li class="bullet"><strong>Appliances:</strong> Black Friday, new model launches, and holiday weekends</li>
                        </ul>
                    </div>
                </div>

                <div>
                    <meta content="6">
                    <h3>6. Don't Let Discounts Drive Unnecessary Purchases</h3>
                    <p>The best way to save money is to not spend it at all. A 70% discount is only valuable if you actually need the item. Buying something you wouldn't have purchased otherwise means you spent money, not saved it, regardless of the discount percentage.</p>
                </div>
