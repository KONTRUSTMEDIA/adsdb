Proximité, Accessibilité et Engagement

Basée à Bezons (95870), dans le Val-d’Oise, RADIOTAMTAM AFRICA est la première webradio africaine en Île-de-France, 100 % en ligne et 100 % indépendante.
Notre mission : informer, divertir et relier la diaspora africaine à travers le monde grâce à une programmation riche en musique, actualités et contenus culturels.

En tant que radio locale connectée à l’international, nous plaçons la proximité avec nos auditeurs au cœur de notre engagement. Chaque émission, chaque playlist et chaque interview est pensée pour créer une connexion humaine, authentique et vivante avec vous.

????️ Une Équipe Talentueuse &amp; des Programmes Diversifiés

Chez RADIOTAMTAM AFRICA, l’énergie et la créativité se ressentent à chaque instant.
Notre équipe de DJ, animateurs, journalistes et chroniqueurs vous offre une expérience audio rafraîchissante, mêlant mix exclusifs, émissions thématiques, reportages originaux et interviews d’artistes émergents de la scène africaine et internationale.

 Des contenus variés : culture, société, musique, politique, innovation, entrepreneuriat et actualité panafricaine.
 Une ambiance unique : « La Parole est une Force », notre devise, incarne la puissance de l’expression libre et du dialogue constructif.

 Une Large Sélection Musicale, 24h/24 et 7j/7

Vous aimez la musique du monde, les rythmes afro, les sons urbains ou les mélodies francophones ?
RADIOTAMTAM AFRICA diffuse une sélection éclectique :

Afrobeat, zouk, coupé-décalé, makossa, rumba, gospel,

Rap, hip-hop français, pop, électro, house, disco, techno,

Classiques et nouveautés en exclusivité.
 Disponible en continu, notre radio offre une expérience sonore haute définition adaptée à toutes les générations et à tous les goûts.

 Accessibilité Universelle – Écoutez-nous Partout !

Grâce à notre technologie de streaming avancée, RADIOTAMTAM AFRICA est accessible dans le monde entier :
 Depuis votre domicile, votre lieu de travail, en voiture ou dans les transports.
 Sur notre site officiel : www.radiotamtam.org

Via notre application iOS gratuite (App Store) et notre application Android (disponible sur Google Play, avec une participation symbolique pour soutenir nos serveurs).

Où que vous soyez, RADIOTAMTAM AFRICA vous accompagne — en Europe, en Afrique, dans les Caraïbes ou au sein de la diaspora mondiale.

Partenaire de Votre Réussite Numérique

Au-delà de la radio, RADIOTAMTAM AFRICA est aussi un acteur du marketing digital et de la communication d’entreprise.
Nous accompagnons les marques, associations et institutions dans :

la rédaction et diffusion de communiqués de presse,

la promotion d’événements culturels et économiques,

la production de contenus multimédias (audio, podcast, vidéo, visuels HD),

le référencement SEO &amp; visibilité locale.

Notre objectif : accroître votre notoriété et générer un impact mesurable grâce à une stratégie sur-mesure fondée sur la créativité et l’innovation.
Nous ne sommes pas une simple agence : nous sommes votre partenaire stratégique sur la voie du succès digital.

 En résumé

 RADIOTAMTAM AFRICA, c’est :

Une radio africaine de proximité, connectée et inspirante ;

Des émissions 100 % culture, musique et société ;

Une présence locale forte à Bezons (95870) et dans le Val-d’Oise ;

Un réseau panafricain qui unit les voix, les talents et les initiatives du continent.

 Choisissez RADIOTAMTAM AFRICA pour une expérience musicale unique, une information libre et panafricaine, et des solutions de communication efficaces adaptées à vos besoins.

    errorTitle = 0;

    $(document).ready(function () {
        // Player Setting

        var listplayer = eval([{"id":"1","typetitrage":"radioking","idserveur":"8","username":"radiotamtam","host":"","port":"0","radiouid":"","mount":"","url":""}]);

        var songMem = new Array();

        playerupdate();
        radioInfoInterval = setInterval(playerupdate, 50000);

        function playerupdate() {
            for (var i = 0; i < listplayer.length; i++) {
                grabRadioInfo(listplayer[i]);
            }
        }


        function grabRadioInfo(player) {
            // Grab Current Song Title
            var D = new Date().toUTCString();
            var utc_timestamp = Date.parse(D);

            $.ajax({
                url: "/cache/titreplayer" + player['id'],
                timeout: 3000,
                type: "GET",
                data: null,
                dataType: "json",
                success: function (data) {
                    var difftime = (utc_timestamp / 1000 - data.mtime);
                    if (isNaN(difftime) || difftime > 30) {
                        $.ajax({
                            type: "GET",
                            url: "/players/index/gettitrageplayer/idplayers/" + player['id'],
                            data: null,
                            timeout: 3000,
                            dataType: "json",
                            success: function (data) {
                                aff_titrage(data, player);
                            },
                            error: function (resultat, statut, erreur) {
                                notifyError();
                            }
                        });
                    } else {
                        aff_titrage(data, player);
                    }
                }
            })
                .fail(function (data) {
                    $.ajax({
                        type: "GET",
                        url: "/players/index/gettitrageplayer/idplayers/" + player['id'],
                        data: null,
                        timeout: 3000,
                        dataType: "json",
                        success: function (data) {
                            aff_titrage(data, player);
                        },
                        error: function (resultat, statut, erreur) {
                            notifyError();
                        }
                    });
                });

            return false;
        }

        function notifyError() {
            errorTitle += 1;

            if (errorTitle > 3) {
                clearInterval(radioInfoInterval);
            }
        }

        function aff_titrage(data, player) {
            if (data.code == "error" || data.title_str == "" || data.title_str == "null" || data.title_str == null) {
                var resultContainer = $('#coverHolder_' + player['id']);
                resultContainer.empty();
                curCover = '<img src="/player/images/noCover.jpg"></img>';
                resultContainer.html(curCover);
                notifyError();
                return;
            } else {
                errorTitle = 0;
            }

            var currentSong = data.title_str;
            var artistSong = data.artist;
            var titleSong = data.title;


            if (currentSong == "" || currentSong == null || currentSong == "null") {
                var resultContainer = $('#coverHolder_' + player['id']);
                resultContainer.empty();
                curCover = '<img src="/player/images/noCover.jpg"></img>';
                resultContainer.html(curCover);

            }

            if (currentSong != songMem[player['id']]) {
                songMem[player['id']] = currentSong;

                $('#artistInfos_' + player['id']).text(artistSong);
                $('#titleInfos_' + player['id']).text(titleSong);

                var image = data.image;

                if (image !== null) {
                    var resultContainer = $('#coverHolder_' + player['id']);
                    resultContainer.empty();
                    curCover = '<img id="coverimg" style="width:100px; height:100px" src="' + image + '" />';
                    resultContainer.html(curCover);
                } else {
                    var resultContainer = $('#coverHolder_' + player['id']);
                    resultContainer.empty();
                    curCover = '<img src="/player/images/noCover.jpg"></img>';
                    resultContainer.html(curCover);
                }
            }
        }

    });
