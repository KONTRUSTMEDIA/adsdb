			<p><strong>Delmas 30 sous tension : deux présumés kidnappeurs abattus par la Police en pleine tentative d’enlèvement cet après-midi</p>
<p>Port-au-Prince, 21 février 2026.- La situation sécuritaire demeure volatile dans la région métropolitaine. Ce samedi, deux individus présentés comme des présumés kidnappeurs ont été abattus par des agents de la Police Nationale d’Haïti (PNH) lors d’une intervention musclée à Delmas 30.</strong></p>
<p>Selon les informations rapportées par Radio Télé Uni, les faits se sont produits alors que les suspects tentaient d’enlever un citoyen dans cette zone réputée sensible. Alertées, les forces de l’ordre seraient intervenues rapidement, empêchant l’exécution du rapt et neutralisant les deux individus armés.</p>
<p>D’après les premiers éléments disponibles, l’opération s’est déroulée en pleine journée, provoquant un mouvement de panique dans le quartier. Plusieurs riverains, surpris par les détonations, ont dû se mettre à l’abri. Aucun blessé civil n’a été signalé jusqu’à présent.</p>
<p>Les autorités policières n’ont pas encore communiqué l’identité des personnes abattues ni précisé si d’autres complices sont activement recherchés. Une enquête serait en cours afin de déterminer les circonstances exactes de l’incident et d’identifier d’éventuelles ramifications criminelles.</p>
<div style="clear:both; margin-top:0em; margin-bottom:1em;"><a href="https://realiteinfo.com/la-police-marque-un-point-decisif-un-dangereux-criminel-tombe-dans-les-filets-des-forces-de-lordre/" target="_blank" rel="dofollow" class="uff4c88cee25f1f9648c8066eb6f9e5d2"><!-- INLINE RELATED POSTS 1/1 //--><style> .uff4c88cee25f1f9648c8066eb6f9e5d2 { padding:0px; margin: 0; padding-top:1em!important; padding-bottom:1em!important; width:100%; display: block; font-weight:bold; background-color:#eaeaea; border:0!important; border-left:4px solid #34495E!important; text-decoration:none; } .uff4c88cee25f1f9648c8066eb6f9e5d2:active, .uff4c88cee25f1f9648c8066eb6f9e5d2:hover { opacity: 1; transition: opacity 250ms; webkit-transition: opacity 250ms; text-decoration:none; } .uff4c88cee25f1f9648c8066eb6f9e5d2 { transition: background-color 250ms; webkit-transition: background-color 250ms; opacity: 1; transition: opacity 250ms; webkit-transition: opacity 250ms; } .uff4c88cee25f1f9648c8066eb6f9e5d2 .ctaText { font-weight:bold; color:#464646; text-decoration:none; font-size: 16px; } .uff4c88cee25f1f9648c8066eb6f9e5d2 .postTitle { color:#000000; text-decoration: underline!important; font-size: 16px; } .uff4c88cee25f1f9648c8066eb6f9e5d2:hover .postTitle { text-decoration: underline!important; } </style><div style="padding-left:1em; padding-right:1em;"><span class="ctaText">LIRE AUSSI:</span>&nbsp; <span class="postTitle">La police marque un point décisif : un dangereux criminel tombe dans les filets des forces de l’ordre</span></div></a></div><p>Présente sur les lieux, Radio Télé Uni a constaté la présence accrue d’agents de la PNH dans la zone après l’intervention, signe d’un dispositif de sécurité renforcé.</p>
<p>Cette nouvelle opération s’inscrit dans un contexte marqué par la recrudescence des actes de kidnapping dans la commune de Delmas, un phénomène qui continue d’alimenter l’inquiétude au sein de la population.</p>
