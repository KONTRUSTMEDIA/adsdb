                <section id="intro">
                    <h2><i class="fas fa-bookmark"></i> 토토사이트</h2>

                    <p>
                        <strong>토토사이트</strong>는 현대 디지털 시대에서 온라인 스포츠 베팅과 다양한 게임을 즐길 수 있는 종합 플랫폼으로 자리매김하고 있습니다.
                        수많은 사용자들이 편리하고 안전한 환경에서 스포츠 경기 결과를 예측하고, 다양한 미니게임을 통해 즐거움을 누리고 있습니다.
                        토토사이트의 가장 큰 장점은 시간과 장소에 구애받지 않고 언제 어디서든 접속하여 베팅을 즐길 수 있다는 점입니다.
                    </p>

                    <p>
                        안전한 토토사이트를 선택하는 것은 사용자 경험의 질을 결정짓는 가장 중요한 요소입니다.
                        검증된 플랫폼은 공정한 게임 운영, 신속한 입출금 처리, 철저한 개인정보 보호를 보장합니다.
                        또한 24시간 고객 지원 서비스를 통해 사용자들의 문의사항을 신속하게 해결하고,
                        다양한 프로모션과 보너스를 제공하여 사용자 만족도를 높이고 있습니다.
                    </p>

                    <div class="highlight-box">
                        <h4><i class="fas fa-lightbulb"></i> 알아두면 좋은 정보</h4>
                        <p>
                            신뢰할 수 있는 토토사이트는 반드시 정식 라이센스를 보유하고 있으며,
                            독립적인 감사 기관의 정기적인 검증을 받습니다. 사이트 선택 시 이러한 인증 마크를 확인하는 것이 중요합니다.
                        </p>
                    </div>

                    <p>
                        최근 토토사이트 시장은 급속도로 성장하고 있으며, 이에 따라 사용자들의 요구도 더욱 다양화되고 있습니다.
                        단순한 스포츠 베팅을 넘어 e스포츠, 가상 스포츠, 라이브 카지노 등 새로운 형태의 게임들이 등장하고 있습니다.
                        이러한 변화에 발맞춰 선도적인 토토사이트들은 최신 기술을 도입하여 사용자 경험을 지속적으로 개선하고 있습니다.
                    </p>

                    <p>
                        모바일 환경의 중요성도 빠르게 증가하고 있습니다. 스마트폰과 태블릿을 통한 접속이 전체 이용자의 70% 이상을 차지하면서,
                        모바일 최적화된 인터페이스와 전용 앱 개발에 대한 투자가 활발히 이루어지고 있습니다.
                        반응형 웹 디자인을 통해 어떤 디바이스에서도 동일한 품질의 서비스를 제공하는 것이 표준이 되었습니다.
                    </p>

                    <p>
                        토토사이트를 이용할 때는 항상 책임감 있는 베팅을 실천해야 합니다.
                        자신의 경제적 상황을 고려하여 적절한 한도를 설정하고, 오락의 목적으로 즐기는 것이 바람직합니다.
                        대부분의 신뢰할 수 있는 사이트들은 자기 제한 도구, 휴식 기간 설정, 베팅 한도 설정 등
                        책임감 있는 게임을 위한 다양한 기능을 제공하고 있습니다.
                    </p>
                </section>

    <script>
        // Mobile Menu Toggle
        function toggleMenu() {
            const navMenu = document.getElementById('navMenu');
            navMenu.classList.toggle('active');
        }

        // FAQ Toggle
        function toggleFaq(element) {
            const faqItem = element.parentElement;
            const allFaqs = document.querySelectorAll('.faq-item');

            allFaqs.forEach(item => {
                if (item !== faqItem) {
                    item.classList.remove('active');
                }
            });

            faqItem.classList.toggle('active');
        }

        // Smooth Scroll
        document.querySelectorAll('a[href^="#"]').forEach(anchor => {
            anchor.addEventListener('click', function (e) {
                e.preventDefault();
                const target = document.querySelector(this.getAttribute('href'));
                if (target) {
                    target.scrollIntoView({
                        behavior: 'smooth',
                        block: 'start'
                    });
                }
                // Close mobile menu if open
                document.getElementById('navMenu').classList.remove('active');
            });
        });

        // Intersection Observer for animations
        const observerOptions = {
            threshold: 0.1,
            rootMargin: '0px 0px -50px 0px'
        };

        const observer = new IntersectionObserver((entries) => {
            entries.forEach(entry => {
                if (entry.isIntersecting) {
                    entry.target.classList.add('animate-fadeInUp');
                }
            });
        }, observerOptions);

        document.querySelectorAll('.info-card, .game-item, .faq-item').forEach(el => {
            observer.observe(el);
        });

        // Header scroll effect
        window.addEventListener('scroll', () => {
            const header = document.querySelector('header');
            if (window.scrollY > 100) {
                header.style.boxShadow = '0 2px 20px rgba(0,0,0,0.3)';
            } else {
                header.style.boxShadow = 'none';
            }
        });
    </script>
